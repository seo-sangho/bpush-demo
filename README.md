## How to use

### 1\. Clone this Repository

```bash
git clone https://github.com/seo-sangho/bpush-demo
```

### 2\. Navigate to the directory

```
cd bpush-demo
```

### 3\. Install dependencies

```
npm install
```

### 5\. Run Dev Server

```
yarn dev
# or
npm run dev
```
